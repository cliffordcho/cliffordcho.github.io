Response

After we sent an XMLHttpRequest object to the server, the server will send a response back in JSON format. While the Computer Vision API documentation does a good job showing what information looks like, it doesn't go into depth on how to access certain parts of the response. We're going to use console.log to analyze the response we get back, and practice accessing certain parts of the response. We'll also create some outs for the rsponse, in case the response takes too long or doesn't return properly.

Reading the Response

Depending on if the server read the response correctly, the API will return a Response number that corresponds to the type of information it sends back. According to this page, there can be responses such as Response 200 (showing the response in JSON format), as well as Response 400, 415, and 500 for different types of errors. For our function, we're going to make it simple: any response that's not 200 will give us an error message. Any response that is 200, will show us the JSON in the console.

We add req.addEventListener to test whether the response status is 200 or not. If it isn't, a simple error message will appear in the console. Otherwise, we're going to parse the JSON response into the variable response. To do that, we want to call responseText on req by writing req.responseText. We also want to parse it using JSON, so we'll write it as JSON.parse(req.responseText) into the variable response. Finally we can print out the response to the console using console.log(response).
